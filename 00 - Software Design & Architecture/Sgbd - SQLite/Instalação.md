# Instalação do SQLite

    1. Baixar o arquivo no site: https://sqlite.org/download.html

        - sqlite-dll-win-x64-3510200.zip
        - sqlite-tools-win-x64-3510200.zip

    2. Criar a pasta com nomle sqlite3
    3. Tirar do zip, colocar na pasta sqlite3
    4. transporte a pasta sqlite3 para o endereço: C:\Program Files (x86)\sqlite3
    5. Vá para Variáveis de Ambiente, em PATH.
    6. Em Novo, coloque C:\Program Files (x86)\sqlite3
